# KNN-for-Dataset-Adult
📌 Descrição

Este projeto aplica um modelo de Machine Learning para classificação utilizando o algoritmo K-Nearest Neighbors (KNN). O objetivo é prever a classe de indivíduos com base em características presentes no dataset.


🎯 Objetivo

Desenvolver um pipeline de aprendizado de máquina envolvendo:

Pré-processamento de dados

Treinamento de modelo

Avaliação de desempenho


📊 Dataset

O dataset utilizado está disponível em:

Kaggle (Adult Income Dataset)


🔽 Como usar

1. Baixe o dataset no Kaggle
   
2. Coloque o arquivo na pasta data/ do projeto


⚙️ Metodologia

O projeto segue as etapas:

1. Pré-processamento
   
Tratamento de valores ausentes

Codificação de variáveis categóricas

Normalização dos dados

2. Modelo

Algoritmo: KNN

Escolha do número de vizinhos (k)

3. Avaliação

Acurácia

Métricas de classificação


💡 Observações

Este projeto demonstra a aplicação prática de um modelo supervisionado, com foco em organização de código e boas práticas para projetos de ciência de dados.
